#  Overview

Memory enables:
- Personalization
- Long-Term Context
- Retrieval-Augmented Generation (RAG)
- Agent Learning
- Knowledge Retention
- Reflection & Planning

# Types of Memory

- Working Memory
- Short-Term Memory
- Episodic Memory
- Semantic Memory
- Procedural Memory
- Long-Term Memory
- Collective Memory

#  Memory Architecture

User → Working Memory → Memory Manager → Vector DB / Knowledge Graph / Relational DB → LLM → Response

#  Memory Lifecycle

Capture → Store → Index → Retrieve → Reason → Update → Forget
